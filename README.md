# Budget Tracker

A simple web application to track your budget, record transactions, and view summaries. Built with **HTML, CSS, and JavaScript**, and runs entirely in the browser.

---

## Features

- Add income and expense transactions  
- View transaction summaries and totals  
- Category-wise breakdown of spending  
- Fully client-side (no backend required)
